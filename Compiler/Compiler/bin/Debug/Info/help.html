Меню текстового редактора

Пункт "Файл" меню текстового редактора

В пункте "Файл" меню текстового редактора реализован следующий функционал (см. рисунок 1):

  • Создание документа
  • Открытие документа
  • Сохранение текущих изменений в документе
  • Сохранение документа в новый файл
  • Выход из текстового редактора


Пункт "Правка" меню текстового редактора

В пункте "Правка" меню текстового редактора реализован следующий функционал (см. рисунок 2):

  • Отмена изменений
  • Повтор последнего изменения
  • Вырезать текстовый фрагмент
  • Копировать текстовый фрагмент
  • Вставить текстовый фрагмент
  • Удалить текстовый фрагмент
  • Выделить все содержимое документа


Пункт "Текст" меню текстового редактора

При вызове команд этого меню должны открываться окна с соответствующей информацией по курсовой работе "Десятичные константы Pascal" (студента Антонянц Егора)

Пункт меню "Текст" содержит следующую информацию (см. рисунок 3):

  • Постановка задачи
  • Грамматика
  • Классификация грамматики
  • Метод анализа
  • Диагностика и нейтрализация ошибок
  • Тестовый пример
  • Список литературы
  • Исходный код программы


Пункт "Пуск" меню текстового редактора

При нажатии на пункт "Пуск" происходит запуск синтаксического анализатора текста (см. рисунок 4).

Приложение имеет справочную систему, запускаемую командой «Вызов справки» (см. рисунок 5).

Справка содержит описание всех реализованных функций меню.(см. рисунок 6)





Пункт "Локализация" меню текстового редактора

Интерфейс позволяет пользователю выбирать язык, на котором он хочет работать (см. рисунок 7, а, б).

Панель инструментов текстового редактора

Панель инструментов содержит кнопки вызова часто используемых пунктов меню:

  • Создание документа (см. рисунок 9, а)
  • Открытие документа (см. рисунок 9, б)
  • Сохранение текущих изменений в документе (см. рисунок 9, в)
  • Отмена изменений (см. рисунок 9,г)
  • Повтор последнего изменения (см. рисунок 9, д)
  • Копировать текстовый фрагмент (см. рисунок 9, е)
  • Вырезать текстовый фрагмент (см. рисунок 9, ж)
  • Вставить текстовый фрагмент (см. рисунок 9, з)
  • Запуск синтаксического анализатора (см. рисунок 9, и)
  • Вызов справки - руководства пользователя (см. рисунок 9, к)
  • Вызов информации о программе (см. рисунок 9, л)


Дополнительные возможности текстового редактора

  1. Интерфейс позволяет работать с несколькими текстовыми документами одновременно (см. рисунок 10, а, б).


  2. Интерфейс позволяет открывать файл при перетаскиваниии его иконки в область редактирования.
  3. Нумерация строк в окне редактирования текста (см. рисунок 11).
